    評分邏輯說明
    工作表現：MIT完成(25%) + 任務完成率(25%) + 專注比(25%) + 工作休息時數效率(25%)
    時數效率：工作≥4h為基準，休息時數對照理想值（每2h配0.5h，4h/8h節點允許更長）
    健康表現：睡眠(33%) + 飲食(33%) + 運動(33%)；睡眠以6–7h為最佳，飲食2101–2300kcal滿分，無運動不扣分（給5分），有運動依項目：高分群10 / 中分群7 / 匹克球5
    人際與情緒：情緒(70%) + 有意義社交(30%)
    今日事件：AI 根據事件豐富度、MIT完成情況、整體品質給分
    連續三天任一維度低於5分會在歷史記錄自動標示警示
    Google Sheets 連動
    透過 Google Apps Script 建立 Web App 端點，每次儲存自動寫入試算表。
    Apps Script 程式碼
    function doPost(e) { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.openById('YOUR_SHEET_ID'); let sheet = ss.getSheetByName('每日復盤'); if (!sheet) { sheet = ss.insertSheet('每日復盤'); sheet.appendRow(['日期','總分','工作','健康','人際情緒','事件', 'MIT','MIT完成','任務率%','專注比%','工作時數','休息時數', '睡眠時數','睡眠分','熱量','運動','情緒','社交']); } sheet.appendRow([ data.date, data.total, data.workScore, data.healthScore, data.socialScore, data.eventScore, data.mit, data.mitDone, data.taskRate, data.focusRate, data.workHours, data.restHours, data.sleepHours, data.sleepScore, data.calories, data.sports, data.mood, data.social ]); return ContentService.createTextOutput('ok'); }
    Web App 網址
    部署設定：執行身份選「我」，存取權選「所有人（包括匿名）」。
